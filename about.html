My Story

I started programming in 2014-2015 on a Raspberry Pi 1B. I played Minecraft on it even though it was just the Raspberry Pi version (which sucked). It made me learn more about computers by breaking a lot of the time. It also had Linux on it, and it was my first computer, so I wasn't complaining. I started programming with Bash, or the Bourne Again Shell. It is by far the easiest programming language because it doesn't have any libraries (good and bad). It allows you to run system commands like cp, sed, grep, rm and more. If you want  to view the manpages (manual pages), run man <command> to view it's page. Bash gives the power to the user by using system commands, which I had been using for a while so I had known about commands so it was fairly easy to get started. I just created trivial programs like one where it asks your name and it says: 'hi, <name>'.

printf "What is your name: "

read -r name

echo "hi $name"

It is simple and you can run it from the shell. But you could put it into a bash script by creating a file with the suffix .sh and adding #!/bin/bash to the top of the file followed by your script. You would run it by typing ./script.sh in the terminal. That was my first experience coding. I also dabbled with python, trying to recreate the ask for name, say hi name but in python2. I used this book if you want to know. I did something like this:

userName = raw_input(“What is your name? “)

print “Welcome to the program,”, userName

Mind you, this was python2, not 3 and raw_input is a terrible way to ask for input because you could inject your own code but I'm getting ahead of myself. I didn't do much python till 2020-2021 as I feel more comfortable with Bash. To this day, Bash is my go to for programming. I've even created a package manager in Bash--the only other package manager written in bash is Portage (for Gentoo), and that is some complex shiz.

I don't really remember much from the journey but I'm here now and as of April 2021, I have gotten so far since 2014.
